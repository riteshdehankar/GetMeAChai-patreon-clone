# ☕ GetMeAChai – Patreon Clone

A full-stack crowdfunding platform where users can support creators by buying them a "chai". Inspired by Patreon.

## 🚀 Features

* 🔐 GitHub Authentication (NextAuth)
* 💳 Razorpay Payment Integration (UPI, GPay, Cards)
* 👤 Dynamic User Pages (`/[username]`)
* 📊 Top Supporters Leaderboard
* 🧾 Payment History
* 🎨 Modern UI with Tailwind CSS

## 🛠️ Tech Stack

* Next.js (App Router)
* MongoDB (Mongoose)
* NextAuth.js
* Razorpay
* Tailwind CSS

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/get-me-a-chai-patreon-clone.git
cd get-me-a-chai-patreon-clone
npm install
npm run dev
```

## 🔑 Environment Variables

Create `.env.local`:

```
MONGO_URI=your_mongodb_uri
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_secret
GITHUB_ID=your_github_id
GITHUB_SECRET=your_github_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```
